# Motel-Managment-System
A simple Motel Management System built using Python and MySQL

## Introduction

This is my Class 12 Computer Science project developed using Python and MySQL.

The project is a simple Motel Management System that helps manage customers, room bookings, and checkouts through a command-line interface.

---

## Features

- Add customer
- Book room
- View customers
- View rooms
- View bookings
- Checkout customer
- Generate invoice
- Remove customer
- Automatic database setup

---

## Technologies Used

- Python
- MySQL
- mysql-connector-python

## How to Run

1. Install Python.
2. Install MySQL.
3. Install the MySQL connector.

```bash
pip install mysql-connector-python
```

4. Open `main.py`.
5. Change the database password.

```python
DB_PASSWORD = "your_mysql_password"
```

6. Run the program.

```bash
python main.py
```

7. Select **Setup Database** from the menu before using the system for the first time.
